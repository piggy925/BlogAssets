1. 使用SSH连接远程服务器

2. 输入以下命令部署宝塔

   ```java
   yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh
   ```
3. 在云服务器实例中添加放行端口`8888`，这里以阿里云为例:

   网络与安全 -> 安全组 -> 配置规则 -> 手动添加

   安装完成后会提示用户名与密码，之后登录宝塔面板时会用到

4. 使用`ip:8888`端口，输入所给用户名密码进入宝塔控制台

5. 然后一键安装服务器程序即可
